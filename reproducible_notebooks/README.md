# Reproducible Notebooks

Diese Notebooks sind unabhängig von Google Colab
und können lokal oder in anderen Jupyter-Umgebungen
ausgeführt werden.

Sie laden die Daten direkt aus dem GitHub-Repository
oder aus öffentlichen Quellen.

Die Notebooks können aber auch gern direkt mit Google Colab geöffnet und ausgeführt werden:
